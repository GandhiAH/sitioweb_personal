# Sitio Web Personal - Gandhi Aguas

**Descripción:**  
Este proyecto es un sitio web personal que contiene información sobre Gandhi Aguas, incluyendo biografía, hobbies y un formulario de contacto funcional hecho en PHP y MySQL.

**Instrucciones de uso:**  
1. Abrir el sitio web en el navegador.  
2. Para enviar un mensaje, ir a `contacto.php`.  
3. Completar Nombre, Correo y Mensaje.  
4. Pulsar "Enviar mensaje".  
5. El mensaje se guardará en la base de datos y aparecerá la confirmación debajo del botón.

**Requisitos:**  
- Servidor web (Apache)  
- Base de datos MySQL
